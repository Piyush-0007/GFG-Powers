Shortcuts

  • Ctrl + ; : Open test case tab.
  • Ctrl + ' : Compile and run.
  • Ctrl + Enter : Submit your code.
  • Alt + Shift + f : Format Code
  • Ctrl + Alt + > or Ctrl + > :
    Add new test case.
  • Ctrl + Alt + < or Ctrl + < :
    Remove the current test case.
